Madrona Decision Support Tool Documentation
############################################

Madrona is a python web application framework for building spatial decision support tools. 
Madrona provides the building blocks to build marine, terrestrial and urban planning applications and allows you to focus on data and models that are important to your decision making process.

.. image:: madrona_outline.png

If your problem involves spatial design, analysis and collaborative decision support, Madrona is the premier tool.
