 28  """Timeout Socket 
 29   
 30  This module enables a timeout mechanism on all TCP connections.  It 
 31  does this by inserting a shim into the socket module.  After this module 
 32  has been imported, all socket creation goes through this shim.  As a 
 33  result, every TCP connection will support a timeout. 
 34   
 35  The beauty of this method is that it immediately and transparently 
 36  enables the entire python library to support timeouts on TCP sockets. 
 37  As an example, if you wanted to SMTP connections to have a 20 second 
 38  timeout: 
 39   
 40      import timeoutsocket 
 41      import smtplib 
 42      timeoutsocket.setDefaultSocketTimeout(20) 
 43   
 44   
 45  The timeout applies to the socket functions that normally block on 
 46  execution:  read, write, connect, and accept.  If any of these  
 47  operations exceeds the specified timeout, the exception Timeout 
 48  will be raised. 
 49   
 50  The default timeout value is set to None.  As a result, importing 
 51  this module does not change the default behavior of a socket.  The 
 52  timeout mechanism only activates when the timeout has been set to 
 53  a numeric value.  (This behavior mimics the behavior of the 
 54  select.select() function.) 
 55   
 56  This module implements two classes: TimeoutSocket and TimeoutFile. 
 57   
 58  The TimeoutSocket class defines a socket-like object that attempts to 
 59  avoid the condition where a socket may block indefinitely.  The 
 60  TimeoutSocket class raises a Timeout exception whenever the 
 61  current operation delays too long.  
 62   
 63  The TimeoutFile class defines a file-like object that uses the TimeoutSocket 
 64  class.  When the makefile() method of TimeoutSocket is called, it returns 
 65  an instance of a TimeoutFile. 
 66   
 67  Each of these objects adds two methods to manage the timeout value: 
 68   
 69      get_timeout()   -->  returns the timeout of the socket or file 
 70      set_timeout()   -->  sets the timeout of the socket or file 
 71   
 72   
 73  As an example, one might use the timeout feature to create httplib 
 74  connections that will timeout after 30 seconds: 
 75   
 76      import timeoutsocket 
 77      import httplib 
 78      H = httplib.HTTP("www.python.org") 
 79      H.sock.set_timeout(30) 
 80   
 81  Note:  When used in this manner, the connect() routine may still 
 82  block because it happens before the timeout is set.  To avoid 
 83  this, use the 'timeoutsocket.setDefaultSocketTimeout()' function. 
 84   
 85  Good Luck! 
 86
