_detect(doc_start, loggedEvents=[], fallback='UTF-8')

source code 

This is the logic from appendix F.1 of the XML 1.0 specification. Pass in the start of a document (>= 256 octets), and receive the encoding to use, or None if there is a problem with the document.

removeDeclaration(x)

source code 

Replace an XML document string's encoding declaration with the same number of spaces. Some XML parsers don't allow the encoding to be overridden, and this is a workaround.
