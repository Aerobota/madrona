Class Land

source code

django.contrib.gis.db.models.Model --+
                                     |
                                    Land

This is where a simplified polygon representation of land gets stored. The greater the number of verticies, the slower the distance analysis so don't get too fancy. Land can be made up of multiple polygons but each polygon gets it's own single polygon record.
