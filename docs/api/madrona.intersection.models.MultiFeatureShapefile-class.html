process_proxy_line(self, field_name='Aj_pct_rck', hard_name='hard', soft_name='soft')

source code 

This function is rather specific to the North Coast MLPA but could concievably be useful elsewhere. This method takes a linear shapefile and, based on the numeric field specified under field_name, splits it into two linear shapefiles. The new shapefiles have segements that are substrings of the original line features. The length of the substrings is determined by the values in the field_name field.
