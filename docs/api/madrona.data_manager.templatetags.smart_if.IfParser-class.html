get_bool_var(self)

source code 

Returns either a variable by itself or a non-boolean operation (such as ``x == 0`` or ``x < 0``).

This is needed to keep correct precedence for boolean operations (i.e. ``x or x == 0`` should be ``x or (x == 0)``, not ``(x or x) == 0``).
