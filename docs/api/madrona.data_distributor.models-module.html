load_potential_targets(modules=settings.INSTALLED_APPS, keep_previous=True, geometry_models_only=True)

source code 

Find all the models in all the apps within the module (madrona) and load their info into the PotentialTarget model and the PotentialTargetField model. If geometry_models_only is True, delete records where there is no potential target field named geometry.
