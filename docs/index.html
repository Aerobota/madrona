Madrona Decision Support Tool Documentation¶

Madrona is a python web application framework for building spatial decision support tools. Originally designed to support the California Marine Life Protection Act Initiative, Madrona has evolved into a reusable platform for marine and terrestrial applications.

If your problem involves spatial data design, spatial analysis and multi-user collaboration, Madrona is the premier tool.
